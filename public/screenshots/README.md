# Screenshots

Place your app screenshots here for the PWA manifest.

## Required files

| File          | Size     | Description                                      |
| ------------- | -------- | ------------------------------------------------ |
| `desktop.png` | 1280×720 | Wide/desktop screenshot (form_factor: "wide")    |
| `mobile.png`  | 390×844  | Narrow/mobile screenshot (form_factor: "narrow") |

## Tips for Microsoft Store submission via PWABuilder

- Desktop screenshot must be at least 1280×720 px
- Mobile screenshot should reflect portrait usage
- Use actual app screenshots showing real UI for best store acceptance
- Run your live URL through https://www.pwabuilder.com to package for the Microsoft Store
